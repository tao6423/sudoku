# Sudoku

A browser-based Sudoku app with puzzle generation, notes mode, conflict highlighting, and undo/redo.

## ▶ [Play Online](https://tao6423.github.io/sudoku/)

## Features

- 4 difficulty levels (Easy / Medium / Hard / Expert)
- Unique puzzle generation with guaranteed single solution
- Conflict highlighting — duplicates in row / column / 3×3 box shown in orange
- Notes mode for penciling in candidates
- Undo / Redo (Ctrl+Z / Ctrl+Y)
- Keyboard navigation (arrow keys + 1–9)
- Mistake counter and timer
